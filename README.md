# Markerless AR Exhibition Sample

이 폴더는 GitHub Pages에 바로 올릴 수 있는 마커 없는 AR 전시관 샘플입니다.

## 배포 방법
1. 이 폴더의 파일을 GitHub 저장소 최상단에 업로드
2. Settings > Pages
3. Source: Deploy from a branch
4. Branch: main / Folder: /(root)
5. 저장 후 배포 주소 접속

## 사용 방법
- 모바일 브라우저에서 배포 주소 열기
- 작품 선택
- "공간에 배치하기" 버튼 탭
- 지원 기기에서 바닥/공간에 배치

## 참고
- Android: WebXR 또는 Scene Viewer로 열릴 수 있음
- iPhone/iPad: Quick Look으로 열릴 수 있음
- 지원되지 않는 기기에서는 3D 미리보기만 동작할 수 있음
